# Sentinel V5: Global Command Center

This is an autonomous AI system for predicting civil unrest risk using a Hybrid LSTM-Transformer model.

## Features
- **Hybrid AI Engine:** Combines LSTM (Trend Analysis) and Transformers (Trigger Detection).
- **Explainable AI (XAI):** Visualizes the exact day and event that triggered the alarm.
- **Generative Briefings:** Uses Llama-3 (via Groq) to generate military-grade situation reports.
